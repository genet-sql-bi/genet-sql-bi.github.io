Brazilian E-Commerce Analytics

A full-stack analysis of the Olist Brazilian e-commerce dataset — connecting PostgreSQL to Python in Google Colab, answering 10 business questions across sales, logistics, customer behavior, and product reviews, then visualizing the complete story in a live Power BI dashboard.

Dataset

Olist (Brazilian E-Commerce)

Scale

100,000+ orders

Stack

PostgreSQL · Python · Power BI

Output

Interactive Dashboard + Notebook

Project Overview

This project takes a real-world e-commerce dataset through the complete analytics lifecycle: raw SQL queries in PostgreSQL, data processing and visualization in Python, and final dashboard delivery in Power BI. The Olist dataset contains transactional data across orders, customers, sellers, products, payments, reviews, and freight logistics from across Brazil.

The goal was to answer 10 concrete business questions that a real operations or strategy team would care about — not just explore the data, but generate actionable answers with supporting visualizations and clear methodology.

10 Business Questions Answered

1Which product categories drive the most sales?
2Which states generate the highest revenue?
3Who are the top-performing sellers?
4What payment methods do customers prefer?
5How does delivery performance compare against estimates?
6What do review scores reveal about product quality?
7What is the average order value across categories?
8How are customers distributed geographically?
9Which categories have the highest freight costs?
10What are the economics of shipping relative to order value?

Tools & Technologies

The project was built across four interconnected tools, each serving a distinct role in the analysis pipeline.

🐘 PostgreSQL 🐍 Python 🐼 Pandas 📊 Matplotlib ☁️ Google Colab 📈 Power BI 🔗 SQLAlchemy 🔌 psycopg2

Project Workflow

1Connect PostgreSQL database to Google Colab via SQLAlchemy and psycopg2
2Load all dataset tables (orders, customers, sellers, products, payments, reviews, freight) into Pandas DataFrames
3Write SQL queries to answer each of the 10 business questions
4Process and aggregate results using Python and Pandas
5Build exploratory visualizations with Matplotlib to validate findings
6Design and publish an interactive dashboard in Power BI for stakeholder delivery
